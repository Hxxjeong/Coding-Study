# [손익분기점](https://www.acmicpc.net/problem/1712)

<br>

## 시간복잡도 분석
-  O(1)

## 코드
```java
package baekjoon.bronze;

import java.io.*;
import java.util.*;

public class b251010_1712_손익분기점 {
	public static void main(String[] args) throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		StringTokenizer st = new StringTokenizer(br.readLine());
		
		int A = Integer.parseInt(st.nextToken());	// 고정비용
		int B = Integer.parseInt(st.nextToken());	// 한개비용
		int C = Integer.parseInt(st.nextToken());	// 판매가격
		
		// A+B*BEP<C*BEP; 총비용 < 총수입이 되는 최초의 판매량
		// A<C*BEP-(B*BEP)
		// A<(C-B)BEP
		
		// A/(C-B) < BEP
		
		// 분모 (C-B)<=0 이면 -1
		// 판매 가격이 가변 비용보다 작거나 같으면 손익분기점 없음
        if (C <= B) {
            System.out.println(-1);
        } else {
            // 손익분기점 계산
            int BEP = A / (C - B) + 1;	// +1해야 이익발생
            System.out.println(BEP);
        }
		
		
	}
}

```
