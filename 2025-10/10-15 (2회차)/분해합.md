# [분해합](https://www.acmicpc.net/problem/1712)

<br>

## 시간복잡도 분석
-  O(N⋅logN)

## 코드
```java
package baekjoon.bronze;

import java.io.*;

public class b251011_2231_분해합 {
	// 생성자 -> 분해합
	//		<-
	public static void main(String[] args) throws IOException{
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int N = Integer.parseInt(br.readLine());

        int result = 0;
        
        // 완전탐색 문제 - 가능한 모든 숫자를 확인하여 조건을 만족하는지 체크
        // 1부터 N까지 모든 숫자를 검사하기
        // 자기자신N은 절대 자신의 생성자가 될 수 없음 : i<N
        for(int i=1;i<N;i++){
            int number = i;    // 현재 돌리고 있는 숫자
            int sum = i;       // 분해합의 시작(본인부터 더하기 시작함)
            
            while(number>0){
                sum += number%10;    // 마지막 자리 수를 더함(남는 나머지가 맨 마지막 수니까)
                number/=10;           // 더한 마지막 자리 수 제거
            }
            
            if(N==sum){
                result=i;
                break;
            }
        }
        System.out.println(result);
	}
	
}


```
